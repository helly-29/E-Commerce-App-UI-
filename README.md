# Flutter E-Commerce App UI (API Integration)

A simple and elegant **E-Commerce UI built with Flutter** that fetches product data from a free online API and displays it in a structured **GridView layout**.

The app showcases product image, name, and price in a responsive grid format.

---

##  Features

-  Fetch products from public API
-  Display product images
-  Show product name and price
-  Responsive GridView layout
-  Smooth UI rendering with API integration

---

##  API Used

- Fake Store API
